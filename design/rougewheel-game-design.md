# RougeWheel - ゲームデザインドキュメント

## 概要

Unity 6.2 で開発するローグライク要素を持つルーレットゲーム。
物理演算によるボール誘導と戦略的なオブジェクト配置を組み合わせた独自のゲーム性を持つ。

## コアコンセプト

通常のルーレットではなく、**縦型ピンボール風アリーナ**にルーレットポケットが並ぶ形式。
プレイヤーは壁やギミックを配置してボールを狙ったポケットへ誘導する。

```
[発射口]
   ↓
┌─────────────┐
│  配置エリア  │  ← ここに壁・ギミックを置く
│   (物理演算) │
│ ○ ○ ○ ○ ○  │  ← ポケット（数字/色）
└─────────────┘
```

## ターンの流れ

1. **賭けフェーズ** — どのポケットに賭けるか決める
2. **配置フェーズ** — 限られたスロット数で壁・ギミックを配置
3. **発射フェーズ** — ボールを発射、物理演算で転がる
4. **結果** — 入ったポケットに応じてコイン獲得/損失

## 配置オブジェクト一覧

| 種類 | 効果 |
|------|------|
| 直線壁 | ボールを反射する基本壁 |
| 曲面壁 | 緩やかに方向転換 |
| ブースター | ボールを加速・方向指定 |
| 磁石 | 特定ポケットに引き寄せる |
| 分裂器 | ボールを2つに分裂させる（両方判定） |

## ローグライク要素

- ステージクリアで**新ギミック解放** or **配置スロット追加**
- **呪い**: 壁が1つ消える、ランダムな位置に障害物が出現
- **ボス**: アリーナに動く壁が追加されるなど特殊ルール
- 所持金が0になるとゲームオーバー（ラン終了）
